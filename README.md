SchoolSphere – School Management System (MERN Stack)

SchoolSphere is a full-stack School Management System built using the MERN stack.
It helps manage Admins, Teachers, and Students with role-based authentication
and a centralized cloud database.

This project was developed as a Final Year Project (FYP) and deployed using
modern cloud platforms.

ENVIRONMENT VARIABLES (Backend)

MONGO_URI=mongodb+srv://<username>:<password>@cluster0.mongodb.net/schoolsphere
JWT_SECRET=your_jwt_secret

Do NOT push .env file to GitHub.

RUN PROJECT LOCALLY

1. Clone Repository
git clone https://github.com/mudassar016/SchoolSphere_FYP.git

2. Backend
cd backend
npm install
npm run dev

3. Frontend
cd frontend
npm install
npm run dev
